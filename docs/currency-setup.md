# Setting the Currency to £ (GBP) in Farming Simulator 25

**For:** Moss Valley Workshop (FS25 Multiplayer)
**Contact:** Joe Heffer

---

## Summary

Farming Simulator 25 has a built-in option to display the currency symbol as £, $, or €. This is a **cosmetic display setting only** — it does not change the actual in-game economy or figures. 1000 units of money is £1000, €1000, or $1000, regardless of which symbol is selected, as there is no real exchange rate applied.

For the Moss Valley workshop, using the built-in setting is recommended over installing a mod, since it avoids adding another version-matching dependency across the 10 client machines.

---

## Option 1: Built-in Currency Symbol Setting (Recommended)

1. Load the Moss Valley savegame.
2. Open the in-game **Settings** menu (General/Gameplay options).
3. Look for the currency/units option — it's grouped alongside similar display settings such as litres vs gallons and hectares vs acres.
4. Select **£ (GBP)** as the currency symbol.

**Important note:** This appears to be a **per-client setting**, not a single server-wide switch. Each participant may need to set this individually on their own machine.

**Action item:** Check and set this on all facilitator machines during the **3rd July training session**, so everyone is consistently displaying £ before the main workshop on **6th July**.

---

## Option 2: "Currency Selector" Mod (Alternative — not recommended for this workshop)

If genuine currency conversion (using real exchange rates) is ever needed rather than just a symbol change, a ModHub mod exists:

- **Mod name:** Currency Selector for Farming Simulator 25
- **Controls:**
  - `Ctrl + Shift + C` — open the currency selection menu
  - `R` (within the menu) — manually set the exchange rate if needed
  - Left/Right arrow keys — fine-tune the currency value
- Supports a wide range of currencies (dollar, euro, ruble, yen, yuan, rupees, and more), with automatic exchange rate conversion applied to balance, land prices, vehicles, and production costs.
- Confirmed compatible with multiplayer, but **all players must have the same version of the mod installed**.

**Why this isn't recommended for the workshop:** Given the existing risks around Moss Valley map installation and version-matching across 10 clients, adding another mod that all participants must install identically increases the chance of connection issues. The built-in setting achieves the desired £ display without this risk.

---

## Recommendation

Use **Option 1** (built-in setting) for the workshop. It requires no additional installation, carries no version-mismatch risk, and is sufficient since the goal is simply to display £ rather than to run a genuine currency conversion.
