# FS25 Modding Resources

**For:** contributors to this mod
**Last checked:** July 2026

---

## Summary

Quick-reference links to the tools and communities relevant to developing and maintaining this mod, with a one-line note on what each is for and how current it is. This is deliberately a short list of essentials, not an exhaustive directory — see each link's own site for full detail.

---

## Official GIANTS Software tools

- **[GIANTS Developer Network (GDN)](https://gdn.giants-software.com/)** — the current hub for all official FS25 modding tools and docs. Supersedes older, scattered GIANTS forum/documentation pages.
- **[GDN Downloads](https://gdn.giants-software.com/downloads.php)** — GIANTS Editor, GIANTS Studio, and the Blender/Maya exporter plugins. Get FS25-specific versions here (currently Editor v10.0.x, Blender exporter v10.0.x) — don't grab the newer 10.5.x builds, which are for GIANTS' other title, Project Motor Racing, not FS25.
- **[GDN Documentation](https://gdn.giants-software.com/documentation.php)** — Lua scripting API reference (LUADOC). Make sure you're reading the **FS25** tab, not the FS22/FS19 legacy docs, since the API differs by version.

This mod is XML/script-based (fillTypes, price data, no custom 3D models), so GIANTS Editor and the Blender/Maya exporters are unlikely to be needed day-to-day — the Lua/XML documentation is the relevant piece.

## Community frameworks

Not directly used by this mod, but useful reference points for how larger, actively-maintained FS25 script mods are structured:

- **[Courseplay_FS25](https://github.com/Courseplay/Courseplay_FS25)** — large AI/automation mod, very active development. Good example of GitHub Actions-based release workflow for FS25 mods.
- **[FS25_AutoDrive](https://github.com/Stephan-S/FS25_AutoDrive)** — autonomous driving routes mod, also active. Has a public Discord for support/dev discussion.

---

## Gaps

No canonical general-purpose FS25 modding wiki or broad community Discord was confirmed as a clear standout during research — the ModHub forums and the frameworks' own Discords above are the most concrete active channels found. If a better general community hub turns up, add it here.
