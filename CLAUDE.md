# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Purpose

A Farming Simulator 25 mod that overrides base-game crop/commodity sell prices to approximate real UK market values. Built for the Moss Valley FS25 multiplayer workshop project.

## Repository layout

- `FS25_UK_Commodity_Prices/` — the mod itself; this whole folder is zipped and shipped as the mod package.
  - `modDesc.xml` — mod metadata (title, description, `<version>`, source file list).
  - `scripts/PriceOverride.lua` — all mod logic: a table of `pricePerLiter` overrides applied at runtime.
  - `icons/modhub/` — DDS icon assets for ModHub submission.
- `icons/` (repo root) — source PNG icons, generated by `scripts/generate_icons.py`.
- `scripts/generate_icons.py` — regenerates the placeholder PNG/DDS icon set; rerun after changing the drawing code.
- `docs/` — contributor and workshop docs; see `docs/README.md` for the index.

## How it works

XML-only fillType price overrides are not applied for non-map mods in FS25, so `PriceOverride.lua` sets `fillType.pricePerLiter` / `startPricePerLiter` at runtime, hooked via `Mission00.loadMission00Finished = Utils.appendedFunction(...)` once the game's economy has finished loading. Prices are stored as £/litre in `UKPrices.PRICES_PER_LITER`, keyed by base-game fillType name (e.g. `WHEAT`, `BARLEY`) — converting from £/tonne requires the fillType's `massPerLiter` density (see comments in the file for each commodity's assumed density).

## Key tooling and verification

- Lua lint: `luacheck FS25_UK_Commodity_Prices/scripts` (config in `.luacheckrc`; std is `lua51`). CI runs this on every push/PR and blocks merges on failure — new globals must be added to `.luacheckrc`.
- XML validation: `xmllint --noout` against every `*.xml` under `FS25_UK_Commodity_Prices`. CI runs this too.
- No automated test suite exists — changes must be tested in-game (load the mod in FS25 and check prices/economy behaviour).
- Icon regeneration: `python scripts/generate_icons.py` (requires Pillow) regenerates both the source PNGs in `icons/` and the DDS files in `FS25_UK_Commodity_Prices/icons/modhub/`.

## Conventions

- When adding/changing a price in `UKPrices.PRICES_PER_LITER`, ground it in a cited real-world UK source (AHDB, DEFRA, etc.) and note the source/density assumption in an inline comment, matching the existing style.
- Verify fillType names and `massPerLiter` densities against the base game's `data/maps/maps_fillTypes.xml` before adding a new commodity.
- Releases are tag-triggered (`vMAJOR.MINOR.PATCH.BUILD`) and must match `<version>` in `modDesc.xml` exactly, or the release workflow fails — see `RELEASING.md`.
